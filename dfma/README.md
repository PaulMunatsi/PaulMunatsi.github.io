# Digital Forensics & Malware Analysis: Zero to Mastery

A complete, self-hosted course site for the DFMA curriculum by **Paul Munatsi**.

Built for GitHub Pages — no server required.

## Structure

```
/
├── index.html          # Course homepage
├── mod00.html          # Module pages (mod00 – mod18)
├── lab00.html          # Lab pages (lab00, lab01–lab16)
├── capstone.html       # Operation Silent Harbour capstone
├── css/style.css       # All styles
├── js/app.js           # All JavaScript
├── images/favicon.svg
└── assets/
    └── markdown/
        ├── modules/    # 19 module .md files
        └── labs/       # 16 lab .md files + capstone
```

## Deploying to GitHub Pages

1. Create a new repository (e.g. `dfma-course`)
2. Push this entire folder as the repo root
3. Go to **Settings → Pages → Source: Deploy from branch → main / (root)**
4. Your site will be live at `https://yourusername.github.io/dfma-course/`
